# Screenplay.md

## Tools & Workflow
- **Nano Banana**: Character consistency across scenes; animated close-ups and interactions.
- **Higgsfield**: Action sequences, cinematic shots, camera moves.
- **Perplexity / Make-A-Video**: Short reveal clips, emotional beats, atmospheric sequences.
- **MidJourney / Stable Diffusion**: Backgrounds, landscapes, establishing shots.

**Workflow Recommendation**:
1. Generate **backgrounds & environments** first.
2. Generate **character animations** via Nano Banana.
3. Overlay **action sequences** using Higgsfield or Perplexity.
4. Merge clips, add cinematic transitions & sound design.
